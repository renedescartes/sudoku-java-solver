Sudoku Solver

About :

This is a simple sudoku solver which can complete missing entries in a sudoku puzzle.

Usage :

  • The puzzle that you need to solve must be placed in a file say “SamplePuzzle.txt”. The entries will be of the following format

X X X X 2 6 X X 9

X 9 X X 4 X X X 6

1 X 5 X 7 X X X X

X X 9 X X 5 X X X

5 3 X X X X X 7 8

X X X 4 X X 2 X X

X X X X 5 X 1 X 3

7 X X X 1 X X 4 X

9 X X 3 8 X X X X


Note the usage of X for entries that are missing. Have only 9 rows and 9 columns with exactly one space to separate the entries.

  • After creating the file you are set to go. Just extract the zip file to some location. You will have a jar file. Execute on the command line,

    java -jar sudoku.jar SamplePuzzle.txt

Alternative Usage :

  • Alternatively you can just open the sudoku.html and solve the puzzle rightaway.
